# Privacy Policy for YT Transcript Grabber

Effective date: July 4, 2026

YT Transcript Grabber is a Chrome extension designed to help users obtain YouTube transcripts through a simple popup interface.

## Overview

YT Transcript Grabber does not collect, store, sell, or share personal data.

The extension only accesses the active browser tab when needed to interact with YouTube pages and send transcript-related requests to a local backend running on `http://localhost:8000`.

## Data Collection

YT Transcript Grabber does **not** collect any of the following:

- Personal information
- Browsing history
- Authentication credentials
- Payment information
- Analytics data
- Tracking data
- Cookies

## Permissions Used

The extension uses the following permissions only for its core functionality:

### `activeTab`
This permission is used to temporarily access the currently active tab when the user interacts with the extension. It is only used to work with the open YouTube page related to transcript extraction.

### Host permissions
The extension uses host permissions for the following domains:

- `https://www.youtube.com/*`
- `http://localhost:8000/*`

These permissions are used only to:
- Access YouTube pages needed for transcript retrieval
- Communicate with the local backend service running on localhost for transcript-related processing

## How Data Is Handled

YT Transcript Grabber does not permanently store user data.

Any interaction with YouTube pages or the local backend is limited to providing the extension's transcript-related functionality. No personal data is used for analytics, profiling, advertising, or tracking.

## Third-Party Services

YT Transcript Grabber does not use third-party analytics, advertising services, tracking technologies, or cookies.

## Data Sharing

YT Transcript Grabber does not sell, transfer, or share user data with third parties.
